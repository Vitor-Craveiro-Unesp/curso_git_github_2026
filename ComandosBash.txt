COMMIT -> SAVE 
ls-> listar arquivo e pastas onde estou.
cd -> mudar de diretorio 
pwd-> caminho absoluto onde estou
ls - a -> acessar arquivos ocultos
clear -> limpar a tela 
mkdir -> criar um diretório
cat-> ver o que ta dentro do arquivo
git init . -> pasta especial permitindo versonamento
rm - rf -> excluir algo forçadamente  
git status -> verificar 
git add nome_arquivo -> sinalizar o que vai entrar no check point 
git commit -m"msg" -> fazer o checkpoint
nano -> chamar editor
git log -> historico dos saves 
rm -> remove diretorio 
git diff -> ver o que esta diferente 
git reset -> alterar se esta em fase de stage para unstage
git checkout -b -> cria uma branch secundária
git branch -> mostra a branch que estou no momento
